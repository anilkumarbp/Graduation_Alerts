# Graduation_Alerts


This is a simple node app to send out daily graduations to Glip Team : 
 
 
## Requirement

 - node v6 and above
 - RingCentral-JS 

## Installation & Setup

### Via npm package
```node
$ git clone https://github.com/anilkumarbp/Graduation_Alerts.git
$ cd Graduation_Alerts
$ npm install
```

### Run the application
